Laboratorio: Ecuación de Segundo Grado con Git y GitHub

Objetivo

Practicar versionamiento de código con Git y GitHub en Java utilizando VS Code como editor.

Escenario 1: Trabajo solo en main

En este escenario se registra el desarrollo progresivo de la solución directamente en la rama principal, documentando cada mejora mediante commits.

Herramientas

Java (JDK)

VS Code

Git

GitHub

Estructura del Proyecto

src/: Código fuente del programa.

.gitignore: Archivos excluidos del repositorio.

README.md: Documentación general.

Instrucciones de Ejecución

Abrir la carpeta en VS Code.

Compilar y ejecutar el archivo Main.java.
